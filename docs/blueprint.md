# **App Name**: SensorViz Pro

## Core Features:

- Data Import: Import sensor data from CSV or text files.
- Interactive Visualization: Generate interactive time series charts for accelerometer and gyroscope data across all three axes.
- Statistical Analysis: Perform basic statistical analysis, calculating mean and peak values for each axis.
- Data Export: Export processed data or charts.

## Style Guidelines:

- Background color: Dark background (#121212) to enhance data visibility.
- Primary color: Vibrant blue (#39A7FF) for interactive elements and emphasis, selected for its techy feel.
- Accent color: Bright orange (#FF8C39) to highlight specific data points and calls to action; its warmth will attract the user's eye without detracting from the visualization.
- Body and headline font: 'Inter', a sans-serif font for clean and modern readability.
- Use minimalistic icons to represent different functions like import, export, and analysis.
- Dashboard layout with clear separation of data visualization, analysis tools, and import/export options.
- Subtle transitions and animations for a smooth user experience when interacting with charts and data.
